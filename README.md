book-shop

## REST API

### 1. Пользователи (/users)
| Метод | Путь | Обработчик | Описание | Аутентификация |
|-------|------|------------|----------|----------------|
| POST | /users | CreateUser | Регистрация нового пользователя | Нет |
| GET | /users | GetUsers | Список пользователей (с пагинацией) | Администратор |
| GET | /users/{id} | GetUser | Профиль конкретного пользователя | Владелец или админ |
| PATCH | /users/{id} | PatchUser | Частичное обновление пользователя | Владелец или админ |
| DELETE | /users/{id} | DeleteUser | Удаление пользователя | Администратор |

### 2. Аутентификация (/auth)
| Метод | Путь | Описание |
|-------|------|----------|
| POST | /auth/login | Получить JWT по email и паролю |
| POST | /auth/refresh | Обновить токен по refresh-токену |

### 3. Книги (/books)
| Метод | Путь | Описание | Аутентификация |
|-------|------|----------|----------------|
| GET | /books | Список книг (фильтр, пагинация) | Нет |
| GET | /books/{id} | Детали книги | Нет |
| POST | /books | Добавить книгу (multipart) | Администратор |
| PATCH | /books/{id} | Редактировать книгу (multipart) | Администратор |
| DELETE | /books/{id} | Удалить книгу | Администратор |

### 4. Адреса доставки (/users/{userId}/addresses)
| Метод | Путь | Описание | Аутентификация |
|-------|------|----------|----------------|
| GET | /users/{userId}/addresses | Список адресов пользователя | Владелец или админ |
| POST | /users/{userId}/addresses | Добавить новый адрес | Владелец |
| GET | /users/{userId}/addresses/{addrId} | Один адрес | Владелец или админ |
| PATCH | /users/{userId}/addresses/{addrId} | Изменить адрес | Владелец |
| DELETE | /users/{userId}/addresses/{addrId} | Удалить адрес | Владелец |

Тело POST: street_address (required), city (required), postal_code (required), country (по умолчанию "Россия"), is_default (boolean).

### 5. Корзина (/cart)
| Метод | Путь | Описание | Аутентификация |
|-------|------|----------|----------------|
| GET | /cart | Получить содержимое корзины текущего пользователя | Обязательна |
| POST | /cart/items | Добавить книгу в корзину | Обязательна |
| PATCH | /cart/items/{itemId} | Изменить количество | Обязательна |
| DELETE | /cart/items/{itemId} | Удалить позицию из корзины | Обязательна |
| DELETE | /cart | Очистить корзину | Обязательна |

Тело POST /cart/items: { "book_id": 42, "quantity": 1 }.
Если такая книга уже есть в корзине, количество суммируется.

### 6. Заказы (/orders)
| Метод | Путь | Описание | Аутентификация |
|-------|------|----------|----------------|
| GET | /orders | Список заказов текущего пользователя | Обязательна |
| GET | /orders/{orderId} | Детали заказа (включая позиции) | Владелец или админ |
| POST | /orders | Оформить заказ из корзины | Обязательна |
